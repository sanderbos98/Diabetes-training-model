Diabetes Prediction Model
This repository contains the code and data for a supervised machine learning project that classifies whether an individual has diabetes based on demographic and clinical features.
Contents

Diabetes_training_model.ipynb — Jupyter notebook containing data preprocessing, model training, and evaluation
diabetes_prediction_dataset.csv — Training dataset sourced from Kaggle.com based on Electronic Health Records (EHR)

Model
A logistic regression model was implemented from scratch using batch gradient descent and cross-entropy loss. The model was trained on 80% of the data and evaluated on a held-out 20% test set.
